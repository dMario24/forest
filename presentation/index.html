1-2. 파이썬 기초 문법 (변수와 자료형)


# 변수 선언
fire_count = 150        # 정수형 (int)
damaged_area = 30.5     # 실수형 (float)
location = "강원도"      # 문자열 (str)
is_active = False       # 불리언 (bool)

# 출력 확인
print(f"발생 지역: {location}")
print(f"피해 면적: {damaged_area} ha")

변수는 데이터를 담는 그릇입니다. 산불 데이터의 속성(면적, 지역 등)을 변수로 표현해봅시다.

리스트(List)와 딕셔너리(Dictionary)


# 리스트: 순서가 있는 데이터 묶음
locations = ["강릉", "삼척", "고성", "동해"]
print(locations[0])  # 강릉

# 딕셔너리: 키(Key)와 값(Value)의 쌍
fire_info = {
    "date": "2023-04-11",
    "loc": "강릉",
    "area": 379
}
print(fire_info["area"])  # 379

조건문 (if-else)

데이터를 필터링하거나 조건을 판단할 때 사용합니다.


area = 150

if area >= 100:
    print("대형 산불입니다.")
    category = "Large"
elif area >= 30:
    print("중형 산불입니다.")
    category = "Medium"
else:
    print("소형 산불입니다.")
    category = "Small"

반복문 (for)

여러 데이터를 순차적으로 처리할 때 사용합니다.


areas = [10, 55, 120, 5]
total_area = 0

for a in areas:
    total_area += a
    if a >= 100:
        print(f"대형 산불 발견: {a} ha")

print(f"총 피해 면적: {total_area} ha")

함수 (Function)

반복되는 코드를 묶어서 재사용성을 높입니다.


def classify_fire(area):
    """피해 면적에 따라 산불 등급을 반환하는 함수"""
    if area >= 100:
        return "대형"
    elif area >= 30:
        return "중형"
    else:
        return "소형"

# 함수 사용
print(classify_fire(120))  # 대형
print(classify_fire(15))   # 소형

1-3. Pandas 기초 (데이터 분석의 핵심)

import pandas as pd

# 가상의 산불 데이터 생성

}

df = pd.DataFrame(data)

Day 2: 데이터 전처리 및 시각화

데이터를 깨끗하게 만들고 인사이트 발견하기

2-1. 데이터 전처리 심화

결측치(Missing Values) 처리

데이터가 비어있는 경우(NaN) 분석 오류를 유발하므로 적절한 처리가 필요합니다.


# 결측치 확인
df.isnull().sum()

# 1. 결측치가 있는 행 제거
df_clean = df.dropna()

# 2. 특정 값으로 채우기 (예: 평균값, 0)
mean_val = df["area"].mean()
df["area"] = df["area"].fillna(mean_val)

이상치(Outlier) 처리

통계적으로 비정상적으로 크거나 작은 값은 분석 결과를 왜곡할 수 있습니다.


# 기술 통계로 이상치 짐작해보기
print(df["area"].describe())

# IQR 방식 등으로 제거하거나,
# 도메인 지식에 따라 별도 그룹으로 분류하여 분석합니다.

데이터 타입 변환

날짜 정보가 문자열(string)로 되어 있다면 날짜형(datetime)으로 변환해야 시계열 분석이 가능합니다.


# 문자열 -> 날짜형 변환
df["date"] = pd.to_datetime(df["date"])

# 월(Month), 요일(Weekday) 추출
df["month"] = df["date"].dt.month
df["weekday"] = df["date"].dt.day_name()

2-2. 기술 통계 및 그룹 분석

Group By: 범주별 통계 산출

지역별, 원인별 산불 통계를 구해봅시다.


# 지역별 피해 면적 합계
area_by_loc = df.groupby("location")["area"].sum()
print(area_by_loc)

# 원인별 발생 건수
count_by_cause = df["cause"].value_counts()
print(count_by_cause)

AI와 함께하는 스마트한 연구 데이터 분석

3-1. 생성형 AI 도구 활용법

ChatGPT / Gemini 활용 원칙

  • 구체적인 지시(Prompt): 막연한 질문보다는 데이터 구조와 원하는 결과를 명확히 설명하세요.
  • 코드 생성 요청: "Pandas로 'location' 컬럼별 'area' 합계를 구하고 막대 그래프를 그리는 코드를 짜줘."
  • 오류 수정(Debugging): 에러 메시지를 그대로 복사해서 붙여넣고 해결책을 물어보세요.
  • 검증 필수: AI가 생성한 코드가 항상 정답은 아닙니다. 반드시 실행하고 결과를 확인해야 합니다.

프롬프트 예시 (연구 벤치마킹)

User: "나는 최근 10년 간의 한국 산불 데이터를 분석하여 '기후 변화가 대형 산불에 미치는 영향'에 대한 소논문을 쓰려고 해. 어떤 변수들을 중점적으로 분석하면 좋을지, 그리고 어떤 시각화가 효과적일지 제안해줘."

AI는 기상 데이터(온도, 습도, 풍속)와 산불 피해 면적 간의 상관분석, 월별 발생 빈도 히트맵 등을 제안해 줄 것입니다.

프롬프트 예시 (코드 작성)

User: "내 DataFrame `df`에는 `date`(날짜), `cause`(원인), `area`(면적) 컬럼이 있어. 원인(`cause`) 중에서 상위 5개만 추려서, 각 원인별 피해 면적 분포를 박스플롯(boxplot)으로 시각화하는 파이썬 코드를 작성해줘. 한글 폰트 설정도 포함해줘."
